# 隐私协议智能合规风险辅助审查系统 - 项目说明

## 一、项目概述

### 1.1 项目名称
隐私协议智能合规风险辅助审查系统

### 1.2 项目定位
面向企业隐私协议、App 隐私政策、网站隐私声明等文本的**辅助审查工具**。

> 注意：系统定位为"辅助审查"，非"自动认定"，更加专业稳妥。

### 1.3 核心功能
- 隐私协议文本上传与解析
- 条款级风险识别
- 法规规则匹配
- 风险等级评估
- 合规依据展示
- 审查报告生成

---

## 二、系统架构

### 2.1 技术架构
```
┌─────────────────┐
│   前端展示层     │  Vue 3 + Element Plus
├─────────────────┤
│   业务逻辑层     │  Spring Boot 3.2
├─────────────────┤
│   数据存储层     │  H2/MySQL
├─────────────────┤
│   规则引擎层     │  合规规则库 + 法规知识库
└─────────────────┘
```

### 2.2 核心业务流程
```
用户上传隐私协议
    ↓
文本清洗与条款切分
    ↓
调用规则引擎匹配
    ↓
结果融合与风险评分
    ↓
生成条款级审查结果
    ↓
关联法律依据与建议
    ↓
前端展示审查报告
```

## 三、功能模块

### 3.1 用户模块
- 用户登录/注册
- 权限管理（USER/ADMIN）

### 3.2 文档模块

### 3.3 分析模块
- 启动分析任务
- 查看分析结果

### 3.4 规则管理模块
- 规则增删改查
- 规则类型管理（KEYWORD/REGEX/MISSING_CHECK）
- 规则启用/禁用

## 六、风险审查项（10 类）

1. 是否明确说明收集的信息类型
2. 是否明确说明收集目的
3. 是否明确说明使用方式
4. 是否说明共享、转让、公开披露规则
5. 是否说明用户权利（查询、更正、删除、撤回同意等）
6. 是否说明未成年人信息保护
7. 是否说明存储期限
8. 是否说明安全保护措施
9. 是否提供联系方式/投诉渠道
10. 是否存在过度收集、模糊授权、一揽子同意等风险表达
